环境

系统：centos 6.5 系统
前端：vue
后端：eggjs

环境搭建

安装 nodejs

设置源

curl -sL https://rpm.nodesource.com/setup_8.x | bash -

安装 nodejs

yum install -y nodejs

安装 nginx

设置源

rpm -ivh http://nginx.org/packages/centos/6/noarch/RPMS/nginx-release-centos-6-0.el6.ngx.noarch.rpm

安装 nginx

yum install nginx

启动 nginx

service nginx start

nginx service 无法执行

vim /etc/selinux/config

SELINUX=enforcing 改为 SELINUX=disabled
重启 reboot

安装 redis

下载文件

wget http://download.redis.io/releases/redis-4.0.8.tar.gz

将 Redis 移动到根目录，方便找

mv redis-4.0.8.tar.gz /etc

解压文件

tar xzf /etc/redis-4.0.8.tar.gz

删掉不需要的

rm -rf /etc/redis-4.0.8.tar.gz

进入 Redis 文件夹

cd redis-4.0.8

生成

make #无法make将上面的4.0.8换成5.0.7

编辑配置文件

vim redis.conf

开始远程连接

#bind 127.0.0.1 //注释

关闭保护模式

preotected-mode no

开启后台模式

daemonize no

启动 redis

src/redis-server redis.conf

打开开机启动配置文件

vim /etc/rc.local

添加 redis 开机启动

/etc/redis-4.0.8/src/redis-server /etc/redis-4.0.8/redis.conf

参考地址：https://www.jianshu.com/p/ea47015dfe3a

程序部署

首先需要安装 git

安装 git

yum install git

创建文件文件夹

mkdir /www/web
mkdir /www/server

部署前端

进入 web 文件夹 clone 前端代码

git clone http://用户名:密码@git地址

部署后端

进入 server 文件夹 clone 后端代码

git clone http://用户名:密码@git地址

进入项目文件夹（如：test），安装依赖

npm install --production

如果报错直接全部安装

npm install

启动后台程序

npm start

配置 nginx 代理

找到配置文件

cd  /etc/nginx/conf.d

编辑配置文件

vim default.conf

添加前端代理

server {
listen 8000;
server_name 127.0.0.1;
root /www/web/test_web; # 前端页面位置

前端界面代理

location / {
    root /www/web/test_web; # 前端页面位置
    try_files $uri $uri/ /index.html; # vue history模式专用
}

前端接口代理

location /api/v1 {
    proxy_pass http://localhost:7001 # 后台地址
}
