Windows上安装Redis和服务，做下记录

下载Reis（版本3.2.100）

我的服务器下载地址

安装

  • 解压下载文件到指定目录

    不需要启动下面3步可以忽略

  • 进入Redis文件reids-server.exe的目录
  • 在此文件夹打开命令行
  • 输入reids-server.exe启动redis

安装服务

进入redis根目录
