Windows开机启动

下载

点击下载

安装

将软件放到nginx目录,并将名称改为nginx-service.exe

新建nginx-service.xml文件

键入内容

<!-- nginx-service.xml -->
<service>
    <id>nginx</id>
    <name>nginx</name>
    <description>nginx</description>
    <logpath>(填写nginx目录，如：D:nginx-1.14.0\)</logpath>
    <logmode>roll</logmode>
    <depend></depend>
    <executable>(填写nginx目录和命令，如：D:nginx-1.14.0\nginx.exe)</executable>
    <stopexecutable>(填写nginx目录和命令，如：D:nginx-1.14.0\nginx.exe -s stop)</stopexecutable>
</service>

安装服务

在nginx文件夹根目录中打开命令行

.\nginx-service.exe install

启动服务

.\nginx-service.exe start

Ubuntu下的操作

下载安装

sudo apt-get install nginx

自动启动服务

sudo systemctl enable nginx #开启服务
sudo systemctl disable nginx #关闭服务

基本命令

sudo systemctl start nginx  #启用nginx服务
nginx #启动nginx
nginx -s stop #停止nginx
nginx -s reload #重启nginx

强行停止

ps -ef | grep nginx #查询进程ID
kill -9 ID #强制停止

常规配置

ssh证书配置

按如下配置即可

server {
    listen 443 ssl;
    server_name  www.xxx.cn; #域名
    ssl_certificate xxx.cn.crt; #下载的证书，放在当前文件夹中
    ssl_certificate_key xxx.cn.key; #下载的证书
    ssl_session_timeout 5m;
    ssl_protocols TLSv1 TLSv1.1 TLSv1.2; #按照这个协议配置
    ssl_ciphers ECDHE-RSA-AES128-GCM-SHA256:HIGH:!aNULL:!MD5:!RC4:!DHE;#按照这个套件配置
    ssl_prefer_server_ciphers on;
    location / {
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_http_version 1.1;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header Host $host;
        proxy_pass http://127.0.0.1:1111; #指向后台程序
        proxy_redirect off;
    }
}

80端口转443端口

之前APP是80端口，现在升级为HTTPS后，post请求变成GET了。
按如下配置判断是否是POST请求，如果是用原来的接口返回，否则转移到443

server {
    listen 80;
    server_name www.xxx.cn;
    location / {
        if ($request_method ~ ^(POST)$) {
            proxy_pass http://127.0.0.1:8765;
            break ;
        }
        return 301 https://www.xxx.cn$request_uri;
    }
}

如果不需要判断POST，直接替换为443即可

server {
    listen 80;
    server_name www.xxx.cn;
    return 301 https://www.xxx.cn$request_uri;
}
