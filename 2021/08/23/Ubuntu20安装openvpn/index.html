Ubuntu20用用脚本安装openvpn

下载脚本

openvpn-install.sh

执行脚本

bash openvpn-install.sh

安装过程中一般都选择默认即可,下面的Client name,可以自定义,保持默认即可

Finally, tell me your name for the client certificate.
Please, use one word only, no special characters.
Client name: client

查看运行状况

正常安装好以后就已经启动好了

ps -ef | grep vpn
nobody     52391       1  0 17:10 ?        00:00:02 /usr/sbin/openvpn --status /run/openvpn-server/status-server.log --status-version 2 --suppress-timestamps --config server.conf
root      106312  104310  0 22:24 pts/0    00:00:00 grep --color=auto vpn

启动或停止

//启动
systemctl start openvpn@server
//停止
systemctl stop openvpn@server

客户端连接

将生成的证书复制出来，可以用来连接
/root/xxx.ovpn
